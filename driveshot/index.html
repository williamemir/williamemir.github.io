Driveshot

This is a simple google chrome extension

It was made with the purpose of testing chrome extension creation with manifest v3.

It takes different types of screenshots of the current tab (area, all page, currrent view), and automatically saves to the user's google drive, or the clipboard, or save to the user's disk, all of this is configurable.

In the options you can set the drive folder id and shared drive id, if they're not set the screenshot is saved to the root of the google drive.
